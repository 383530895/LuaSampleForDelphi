LuaSampleForDelphi
==================
# 使用ライブラリ
以下のライブラリを使用しています

* [Lua for Windows](http://luaforwindows.luaforge.net/)
* [Lua Unit for Delphi2009](http://aoikujira.com/demo/hakkaku/rc/20090414~f3YTG-luaunit_delphi2009.zip)

lua.dll は Lua for Windows に同梱されている lua5.1.dll をリネームして使用しています


# 開発環境
以下の環境で開発しました

* Windows Vista SP2
* Turbo Delphi 2006
